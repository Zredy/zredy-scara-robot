# zredy-scara-robot
A "SCARA" prototype robot

Programmed and controlled with a Arduino Nano, and Processing on PC
CAD Design done in Fusion 360

Mini Gripper (not mine) on Thingiverse: https://www.thingiverse.com/thing:2415
